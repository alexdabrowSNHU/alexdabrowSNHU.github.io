Personal Reflection and Assessment

When I came into the Computer Science program at SNHU, the only real skill or knowledge I had regarding the world of software development was printing “Hello, World!” in Python.


IT-140 (Intro to Scripting) was the first class where instead of just printing a statement in Python, it was about defining and using functions, learning about return values, using variables and dictionaries to create a console game.

IT-145 (Intro to App Development) was my first introduction to Java. This is where we developed a console application for the animal rescue management system that this capstone is based on.

CS-300 (Data Structures and Algorithms) gave me one of my first introductions to writing in C++ and learning how to build sort algorithms.

CS-360 (Mobile Architecture and Programming) was my first experience doing programming for mobile in Android Studio, as well as implementing a login system.

CS-405 (Secure Coding) was about secure coding, learning about common vulnerabilities, static testing methods, and mitigation of threats with Defense in Depth.

CS-465 & CS-470 (Full Stack Development I & II) was about learning the MEAN stack (MongoDB, Express, Angular, and Node.js), as well as integrating AWS services like Lambda and API Gateway.


These classes, along with the rest of the program, have given me a solid foundation in software development. I have learned how to use Git and GitHub, how to write code in multiple languages, how to develop secure applications, and communicate ideas, concepts and strategies to technical and non-technical audiences alike.


This all culminates into this final capstone. I did not want to rely on things I learned. I wanted this capstone to be another experience in learning, not just exercising what I already know. That is why I chose to use React for the frontend, and Spring Boot on Azure App Service on the backend. I think it is an important skill to be able to quickly learn and adapt to new tools and use them proficiently, which is what I believe I have done with this capstone project.
